## 3. PWA (installable + hors-ligne)

- `public/manifest.webmanifest` : nom, `display: standalone`, portrait, couleurs `#05070f`, icônes.
- Icônes générées : `icon-192/512`, `icon-maskable-512` (padding adapté), `apple-touch-icon` (180), `favicon-32` ; source 1024 dans `docs/design/`.
- `public/sw.js` : service worker écrit à la main —
  - pages HTML : réseau d'abord, repli cache (mode hors-ligne) ;
  - assets hashés Vite : cache d'abord ;
  - **aucune** interception des appels API externes (Supabase/auth jamais cachés).
- Enregistrement dans `__root.tsx` (prod uniquement, `import.meta.env.DEV` ignoré).
- Balises meta : `theme-color` alignée sur le fond réel (`#05070f`), `apple-mobile-web-app-*`.
